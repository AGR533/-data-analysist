## Dataset
Telecom Customer Churn (Selected Columns)

## What This Notebook Covers
- Dataset description and variable types
- Data quality checks (missing values, duplicates, inconsistent categories, outliers, wrong formats)
- Basic statistics (mean, median, min, max, category counts)
- Data visualizations (bar charts, histogram)
- Final conclusion and insights

## Libraries Used
- pandas
- matplotlib

3. Upload the CSV file
